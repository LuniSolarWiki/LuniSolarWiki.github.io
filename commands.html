Server Commands

A list of essential commands you'll need to navigate and thrive on Lunisolar.

Island Commands

These commands control your OneBlock island.

Command Description
/ob create Creates your new OneBlock island.
/ob go Teleports you to your island's home position.
/ob trust {player} Allows a player to build and break on your island. (Be careful!)

Economy Commands

Commands for making money, selling, and trading.

Command Description
/sell Sells items from your inventory (often used with /sell all).
/ah Opens the Auction House to buy items from other players.
/ah sell Lists the item currently in your hand for sale.

Teleport & Warps

How to get around the server and to other islands.

Command Description
/warp crates Teleports you to the server's crate area.
/warp pvp Teleports you to the PvP arena and KoTH.
/pwarp create {name} Creates a public warp on your island for other players.
/pwarp {name} Teleports you to another player's public warp.
/pwarp manage {name} Manages the settings for your player warp.
/pwarp remove {name} Deletes one of your player warps.

Minigame & Event Commands

Commands for server games and events.

Command Description
/pg join Opens the menu to join Party Games (BlockParty, Spleef, etc.).
/duel {player} Challenges another player to a 1v1 duel.

Other Commands

Useful commands for help and voting.

Command Description
/vote Opens the in-game list of vote links to get keys.
/helpop {message} Sends a private message to all online staff members.
